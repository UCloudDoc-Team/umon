

# 消息订阅

## 1.消息订阅简介

消息订阅产品化各产品实例和平台服务生命周期与运转中的重要消息，构建完善的消息消费渠道与流程，支撑客户云上监控与运维。所提供的可订阅的消息，由内部各产品模块与底层基础设施服务获取，经过聚合、判定和收敛再最终呈现。
